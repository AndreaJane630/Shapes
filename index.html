Creating Triangles in CSS

Do you see how we derived the triangles? Each one of the 4 is created using 3 borders: 1 with color and the other 2 transparent. Take a look at the 2nd box in the 1st row. It has no height or width, but it is made up of 4 borders. Although each border is a rectangle, we don't see all of it. Using different border colors enables you to see this - if we had used the same color for all, we'd have a solid colored div. In both boxes, the corners are mitered (see the image).

So that you can see how this is done, let's look at the 1st triangle. It is formed by a left border, and by top and bottom borders (look at the "empty box"). The empty box is 150px x 150px. So, the left border of the triangle will be 75px. This is the thickness of the left border and determines the height of the triangle (in this case, going from left to right, as this triangle is "on its side"). The top and bottom borders will be 75px as well. Added together, we get the base of the triangle. We're just using these dimensions based on the empty box in the 1st row. I outlined the triangles in the 1st row so that you get an idea of the 3 borders involved. If you want triangles that are narrower, thicker, etc. you will adjust the pixels accordingly, once you understand the basic recipe. The empty box is just a way to understand what's going on. Use it as a reference point when you get confused.
